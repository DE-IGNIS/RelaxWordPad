# RelaxWordPad

RelaxWordPad is a lightweight Notepad-inspired text editor built using Java. It focuses on simplicity, speed, and a distraction free writing experience, with a small touch of personalization through subtle color variations.

---

## ✨ Features

* 📝 Clean and minimal text editing interface
* ⚡ Fast and lightweight performance
* 🎨 Subtle color theme options (based on mood)
* 📂 Basic file operations (Create, Open, Save)

---

## 🛠️ Tech Stack

* **Language:** Java
* **Framework/Library:** Java Swing / AWT (for GUI)

---

## 🚀 Getting Started

### Prerequisites

* Java JDK (8 or above)
* Any Java IDE (IntelliJ IDEA, Eclipse, VS Code)

### Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/RelaxWordPad.git
   ```

2. Navigate to the project directory:

   ```bash
   cd RelaxWordPad
   ```

3. Compile and run:

   ```bash
   javac Main.java
   java Main
   ```
